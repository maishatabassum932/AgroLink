# AgroLink

## Overview

AgroLink is a full-stack web application developed as part of my Internship/Practicum program. The platform connects farmers and consumers directly, enabling users to browse agricultural products, place orders, and support a transparent agricultural marketplace.

## Features

- User Registration and Login
- Secure Authentication
- Product Browsing
- Product Search and Filtering
- Shopping Cart Management
- Order Placement
- Farmer Product Management
- Admin Dashboard
- Order Monitoring and Management

## Technologies Used

### Frontend
- React.js
- HTML5
- Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Project Objectives

- Connect farmers and consumers through a digital platform
- Simplify agricultural product management
- Improve transparency in product purchasing
- Provide an efficient online marketplace experience

## Learning Outcomes

Through this project, I gained practical experience in:

- MERN Stack Development
- RESTful API Development
- Database Design and Integration
- Authentication and Authorization
- Git and GitHub Version Control
- Full-Stack Web Application Development
